# Moldes em A4 - Conversor de PDF

Aplicação web profissional para converter moldes em PDF de grandes formatos (CAD, Audaces, etc.) em páginas A4 prontas para impressão em impressoras comuns.

## 🚀 Tecnologias

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)
- [pdf-lib](https://pdf-lib.js.org/)

## 📦 Funcionalidades

- **Divisão Automática**: Converte PDFs de grandes formatos em uma grade de páginas A4.
- **Marcas de Corte**: Adiciona guias visuais para facilitar a montagem do molde após a impressão.
- **Processamento Local**: Os arquivos são processados diretamente no navegador, garantindo privacidade e segurança.
- **Preview em Tempo Real**: Visualize como o molde será dividido antes de baixar.

## 🛠️ Instalação e Uso Local

1. Clone o repositório:
   ```sh
   git clone https://github.com/seu-usuario/convertermoldesa4.git
   ```

2. Instale as dependências:
   ```sh
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```sh
   npm run dev
   ```

4. Para gerar a versão de produção:
   ```sh
   npm run build
   ```

## 🌐 Deploy

O projeto está configurado para deploy automático no **GitHub Pages** via GitHub Actions. Sempre que um push for feito na branch `main`, o workflow será executado e a versão atualizada será publicada.
